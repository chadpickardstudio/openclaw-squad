# 04-ops-manager

**Department:** C-Suite
**Org Title:** COO
**Appears in:** 3 of 5 scenarios (common)

## Role

Process management and operational coordination. Handles scheduling, admin tasks, vendor management, document preparation, and process documentation. The glue that keeps day-to-day operations running in process-heavy businesses.

## Core Tasks

- Scheduling and calendar coordination
- Administrative task management
- Vendor and supplier coordination
- Document preparation and template population
- Process documentation and SOP maintenance
- Deadline tracking and follow-ups
