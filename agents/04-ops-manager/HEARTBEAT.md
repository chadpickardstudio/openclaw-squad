# HEARTBEAT.md — Ops Manager

## Triggers

### Daily (7:30 AM) — Operations Check
- Review today's deadlines and scheduled tasks
- Check for pending admin requests from other agents
- Surface any vendor renewals due within 14 days
- Update task completion status

### Weekly (Monday 8:00 AM) — Operational Review
- Compile weekly operations summary for Lead
- Review all active processes for bottlenecks
- Check SOP currency — any need updating?
- Plan this week's operational priorities

### Monthly (Last day of month) — Vendor & Process Review
- Review all vendor contracts and costs
- Assess process efficiency — any improvements to propose?
- Update master deadline calendar for next month
- Send monthly ops report to Lead

## HEARTBEAT_OK Discipline

- Don't micromanage other agents' tasks — handle admin, not oversight
- Batch non-urgent admin items into daily processing
- Smart silence > unnecessary status updates
