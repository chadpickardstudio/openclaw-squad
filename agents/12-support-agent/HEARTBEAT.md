# HEARTBEAT.md — Support Agent

## Triggers

### Every 30 Minutes — Inbox Check
- Check for new customer inquiries
- Respond to pending tickets
- Update ticket statuses

### Daily (End of Day) — Support Summary
- Compile daily ticket summary (opened, resolved, escalated)
- Identify any unresolved tickets needing follow-up
- Log recurring patterns

### Weekly (Friday) — Pattern Report
- Compile weekly support patterns for Lead
- Identify top 3 recurring issues
- Propose KB additions or product improvements
- Report customer sentiment trends to Client Manager

## HEARTBEAT_OK Discipline

- Prioritize speed of response over thoroughness of reporting
- Don't over-report to Lead — batch insights into weekly pattern report
- Smart silence on internal channels > constant status updates
