# 12-support-agent

**Department:** Operations
**Org Title:** Customer Support
**Appears in:** 3 of 5 scenarios (common)

## Role

Frontline customer support and inquiry handling. Manages inbound questions, ticket routing, FAQ responses, booking confirmations, and first-line issue resolution. The first touchpoint for customer-facing businesses.

## Core Tasks

- Customer inquiry response and triage
- FAQ and knowledge base lookups
- Ticket creation and routing to specialists
- Booking and appointment confirmations
- Waitlist management
- First-line issue resolution
- Escalation to Client Manager or Lead when needed

## Squad Presence

- **The Operator (Tier 1):** No
- **The Department (Tier 2):** Yes — frontline support
- **The Firm (Tier 3):** Yes
