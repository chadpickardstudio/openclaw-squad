# HEARTBEAT.md — Data Analyst

## Triggers

### Daily (7:00 AM) — Metrics Check
- Check for anomalies in key metrics (>20% deviation)
- Alert Lead immediately if anomaly detected
- Update daily tracking data

### Weekly (Monday 8:00 AM) — Weekly Dashboard
- Compile weekly KPI dashboard
- Compare against benchmarks and targets
- Identify trends and patterns
- Send weekly dashboard to Lead and Strategist

### Monthly (1st of month) — Monthly Deep Dive
- Full channel performance report with attribution
- Funnel analysis with conversion rates by stage
- Monthly trend analysis and forecast
- Benchmark review — update targets if needed
- Send monthly report to Lead

## HEARTBEAT_OK Discipline

- Don't report metrics that haven't changed — focus on movements
- Batch routine data into scheduled reports
- Only interrupt for genuine anomalies
