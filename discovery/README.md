# Discovery

Client onboarding pipeline — from first conversation to live squad.

## Documents

| # | Document | Purpose |
|---|----------|---------|
| 01 | [Assessment Questionnaire](01-assessment-questionnaire.md) | Structured conversation guide for the initial client meeting (15–20 min) |
| 02 | [Assessment-to-Squad Router](02-router.md) | Decision logic that converts assessment answers into a deployment specification |
| 03 | [Client Onboarding Packet](03-onboarding-packet.md) | Client-facing welcome document delivered after squad deployment |

## Pipeline

```
Client says "I'm interested"
        ↓
01 — Assessment Questionnaire
     captures business, pain points, tools, budget, brand
        ↓
02 — Assessment-to-Squad Router
     selects template, activates agents, assigns skills, routes models
        ↓
research/23 — Deployment Playbook executes the spec
research/24 — Skills Blueprint generates the skills
research/25 — Testing Checklist validates everything
        ↓
03 — Client Onboarding Packet
     tells the client how to work with their new squad
        ↓
Squad is live
```
