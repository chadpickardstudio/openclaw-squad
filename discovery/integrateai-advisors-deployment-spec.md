## Tool Stack Recommendations

These tools need to be selected before deployment. Recommendations based on solo founder + AI-first workflow:

| Function | Recommended | Why | Monthly Cost |
|----------|-------------|-----|-------------|
| **CRM** | HubSpot Free | Free tier covers contact/deal management, good API, scales well | £0 |
| **Email marketing** | Beehiiv or MailerLite | Beehiiv: great for newsletter-first, free up to 2.5k subs. MailerLite: better automation, free up to 1k subs | £0–15 |
| **Social scheduling** | Buffer or Later | Both have free tiers, both have APIs for automation | £0–15 |
| **Blog/CMS** | Ghost (self-hosted) or WordPress | Ghost: clean, API-first, self-host on same VPS. WordPress: more plugins, wider ecosystem | £0–5 |
| **Analytics** | Google Analytics + Search Console | Free, industry standard, excellent API | £0 |
| **Communication** | Slack (free tier) | Squad internal comms, channel-per-function | £0 |
| **Domain email** | Google Workspace or Zoho Mail | Professional @chadpickard.studio emails | £5–12 |
| **Calendar/Booking** | Cal.com (free) or Calendly (free) | Booking links for discovery calls | £0 |

**Estimated tool stack cost: £5–47/month** (depending on selections and tiers)
