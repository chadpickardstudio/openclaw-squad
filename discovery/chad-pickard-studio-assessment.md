## Router Decision

Based on this assessment:

- **Pain points:** 4 critical + 3 major = full squad required
- **Tools:** Greenfield — everything needs to be set up
- **Budget:** Uncapped
- **Timeline:** Everything operational from day one
- **Scale:** Solo founder, pre-revenue, building the business AND using the product simultaneously

### Recommended Squad Template: Custom / Full Department+

This is not a standard client deployment. This is **dogfooding** — Chad Pickard Studio using OpenClaw to build itself. The squad needs to cover:

1. **Lead** — CEO agent, routes all work, manages the squad
2. **Content Creator** — Blog, copy, long-form content
3. **Social Media Manager** — All platforms, daily posting, engagement
4. **Email Marketer** — Sequences, newsletters, list building
5. **Market Researcher** — SEO, competitor analysis, audience insights
6. **Sales Rep** — Outbound prospecting, pipeline, follow-up
7. **Strategist** — Campaign planning, growth roadmap, analytics
8. **Data Analyst** — Performance tracking, reporting, insights

**Minimum viable squad: 8 agents (Lead + 7 specialists)**

This is larger than a typical "Department" (Lead + 4-6) because the business literally has zero existing marketing/sales infrastructure. Every function must be built from scratch.
