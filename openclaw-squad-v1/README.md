# OpenClaw Squad v1

Multi-agent team built with OpenClaw (2026).

## Structure

- `/agents/` — **Single source of truth** for every elite agent role, grouped by department.
  Each department contains a `department-blueprint.md` and numbered role subfolders.
  Each role has 8 elite files (IDENTITY, SOUL, skills, STYLE, GOALS, CONSTRAINTS, EVOLUTION, HEARTBEAT).
- `/squad-blueprints/` — Custom client mixes
- `/research/` — 21 deep-dive topics
- `/decisions/` — Architectural Decision Records
- `/spec/` — Living Specification
