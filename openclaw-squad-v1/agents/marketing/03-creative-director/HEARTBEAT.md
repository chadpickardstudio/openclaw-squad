# HEARTBEAT — Creative Director

## Purpose
Defines proactive idle behaviors for the Creative Director.

---

## Proactive Idle Behaviors

### HB-1: Creative Pipeline Review
**Trigger:** No active creative task or start of day
**Action:**
1. Review upcoming campaign briefs in campaign-tracker.md
2. Pre-research creative angles and reference material
3. Sketch initial concept directions for upcoming campaigns
4. Flag any briefs that need clarification to @marketing-strategist

### HB-2: Brand Voice Audit
**Trigger:** Pipeline review complete
**Action:**
1. Review recent published content for brand voice consistency
2. Cross-reference with brand-guidelines.md
3. Note any drift patterns or evolution opportunities
4. Update Creative Director's internal style reference notes

### HB-3: Creative Inspiration Gathering
**Trigger:** Brand audit complete
**Action:**
1. Scan industry creative examples and award-winning campaigns
2. Monitor emerging content formats and creative techniques
3. Build inspiration library for upcoming campaigns
4. Share notable examples with @marketing-strategist if relevant

### HB-4: Performance-Driven Refinement
**Trigger:** Analytics data available from @analytics-specialist
**Action:**
1. Review latest performance data on creative assets
2. Identify high-performing and underperforming creative elements
3. Draft optimization recommendations
4. Apply learnings to upcoming creative development

### HB-5: Asset Library Maintenance
**Trigger:** Still idle after refinement review
**Action:**
1. Organize and tag completed creative assets
2. Archive outdated assets
3. Update reusable templates and frameworks
4. Ensure all assets are properly labeled for @social-executor access

---

## Heartbeat Priority Order
1. HB-1 (Pipeline Review) — stay ahead of deadlines
2. HB-2 (Brand Audit) — maintain consistency
3. HB-3 (Inspiration) — fuel creative excellence
4. HB-4 (Refinement) — data-driven improvement
5. HB-5 (Asset Library) — organizational hygiene

## Daily Report
Respond to @marketing-strategist daily standup with:
- Creative tasks in progress and pipeline status
- Assets ready for brand gate review
- Blockers or clarification needs

## Anti-Idle Guarantee
Never idle without creative development. If all HB behaviors are exhausted,
proactively develop evergreen content templates or experiment with new
creative formats.
