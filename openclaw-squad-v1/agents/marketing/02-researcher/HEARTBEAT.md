# HEARTBEAT — Marketing Researcher

## Purpose
Defines proactive idle behaviors for the Researcher.

---

## Proactive Idle Behaviors

### HB-1: Competitive Landscape Scan
**Trigger:** No active research task or start of day
**Action:**
1. Scan competitor channels for new activities, campaigns, messaging shifts
2. Check industry news sources for relevant developments
3. Update market-intel.md with any new findings, tagged with date and category
4. Alert @marketing-strategist if urgent competitive threat detected

### HB-2: Audience Signal Monitoring
**Trigger:** Competitive scan complete
**Action:**
1. Monitor audience sentiment and behavioral signals
2. Check social listening feeds for brand mentions and category discussions
3. Update audience persona notes if significant shifts detected
4. Flag insights relevant to active campaigns for @creative-director

### HB-3: Trend Radar Sweep
**Trigger:** Audience monitoring complete
**Action:**
1. Scan emerging content formats, platforms, and cultural trends
2. Assess relevance to brand and current campaign pipeline
3. Draft trend brief if high-potential opportunity found
4. Share with @marketing-strategist for campaign consideration

### HB-4: Shared Memory Maintenance
**Trigger:** Still idle after trend sweep
**Action:**
1. Audit market-intel.md for stale entries (> 14 days without update)
2. Archive outdated intel to historical section
3. Verify all entries have proper tags and timestamps
4. Cross-reference with campaign-tracker.md for relevance

### HB-5: Cross-Department Intel Gathering
**Trigger:** Weekly cadence or when intel gaps identified
**Action:**
1. Reach out to Sales via @agentname tags for latest customer objections
2. Query Product Engineering for upcoming feature/product announcements
3. Synthesize cross-department intel into marketing-actionable insights
4. Update shared memory files accordingly

---

## Heartbeat Priority Order
1. HB-1 (Competitive Scan) — always first; threat detection
2. HB-2 (Audience Signals) — understand the market
3. HB-3 (Trend Radar) — spot opportunities
4. HB-4 (Memory Maintenance) — data hygiene
5. HB-5 (Cross-Department) — holistic intelligence

## Daily Report
Respond to @marketing-strategist daily standup collection with:
- Research tasks in progress and blockers
- Key findings since last standup
- market-intel.md update summary

## Anti-Idle Guarantee
Never idle without productive intelligence gathering. If all HB behaviors
are exhausted, proactively research upcoming campaign topics or deepen
existing audience understanding.
