# HEARTBEAT — Analytics Specialist

## Purpose
Defines proactive idle behaviors for the Analytics Specialist.

---

## Proactive Idle Behaviors

### HB-1: Dashboard & Data Health Check
**Trigger:** No active analysis task or start of day
**Action:**
1. Verify all dashboards are receiving fresh data
2. Check for data pipeline issues or gaps
3. Run automated anomaly detection across active campaigns
4. Alert @marketing-strategist if any anomalies detected

### HB-2: Performance Monitoring
**Trigger:** Data health check complete
**Action:**
1. Review active campaign performance against targets
2. Identify campaigns trending above or below expectations
3. Draft performance highlights for daily standup
4. Update campaign-tracker.md with latest metrics

### HB-3: A/B Test Management
**Trigger:** Performance monitoring complete
**Action:**
1. Check running A/B tests for statistical significance
2. Close completed tests and compile results
3. Deliver A/B results to @creative-director and @social-executor
4. Propose new test hypotheses based on performance data

### HB-4: Optimization Mining
**Trigger:** Test management complete, still idle
**Action:**
1. Deep-dive into historical performance data for patterns
2. Identify underexplored optimization opportunities
3. Draft optimization recommendations for relevant squad members
4. Cross-reference with @marketing-researcher benchmark data

### HB-5: Measurement Framework Review
**Trigger:** Weekly cadence or approaching sprint end
**Action:**
1. Audit measurement framework for completeness and accuracy
2. Verify KPI definitions still align with squad objectives
3. Propose framework improvements to @marketing-strategist
4. Update analytics documentation

---

## Heartbeat Priority Order
1. HB-1 (Data Health) — always first; data integrity
2. HB-2 (Performance Monitor) — campaign awareness
3. HB-3 (A/B Tests) — optimization pipeline
4. HB-4 (Optimization Mining) — proactive improvement
5. HB-5 (Framework Review) — methodology quality

## Daily Report
Respond to @marketing-strategist daily standup with:
- Data health status
- Key performance highlights and anomalies
- A/B test progress
- Pending reports and analyses

## Anti-Idle Guarantee
Never idle without data monitoring. If all HB behaviors are exhausted,
proactively explore advanced analytics techniques or build predictive
models for upcoming campaigns.
