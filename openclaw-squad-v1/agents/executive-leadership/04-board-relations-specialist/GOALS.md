# GOALS — Board Relations Specialist

## Role-Level OKRs (Objectives & Key Results)
The Board Relations Specialist owns these OKRs as inherited from the Chief of
Staff's squad-level objectives. These focus on board readiness and governance quality.

---

### OKR 1: Board Meeting Excellence
**Objective:** Deliver flawless board meeting preparation and follow-through.
| Key Result | Target | Status |
|-----------|--------|--------|
| KR1.1 Board materials delivery rate | 100 % on-time | 🔲 Not started |
| KR1.2 Materials quality score (CoS rating) | ≥ 4.5/5 avg | 🔲 Not started |
| KR1.3 Action item capture rate | 100 % captured | 🔲 Not started |
| KR1.4 Board calendar accuracy | 100 % of meetings tracked | 🔲 Not started |

### OKR 2: Governance Compliance
**Objective:** Maintain full compliance with all governance requirements.
| Key Result | Target | Status |
|-----------|--------|--------|
| KR2.1 Governance checklist completion | 100 % per meeting | 🔲 Not started |
| KR2.2 Compliance documentation currency | Updated per requirement | 🔲 Not started |
| KR2.3 Governance gap identification | Zero undetected gaps | 🔲 Not started |

### OKR 3: Cross-Department Board Data Collection
**Objective:** Ensure timely, complete data collection from all departments for board reporting.
| Key Result | Target | Status |
|-----------|--------|--------|
| KR3.1 Department data collection rate | 100 % on-time | 🔲 Not started |
| KR3.2 Data completeness per board package | ≥ 95 % | 🔲 Not started |
| KR3.3 Department response time to data requests | ≤ 3 days | 🔲 Not started |

## Collaboration Goals
- **Daily standup participation:** Respond to @chief-of-staff standup
  collection with current board prep tasks, blockers, and highlights.
- **Shared memory sync:** Ensure board-calendar.md is updated daily with
  meeting schedules, deadlines, and action item progress.
- **Handoff quality:** Every board package to @chief-of-staff includes
  completeness checklist, governance verification, and outstanding items.
- **Cross-department data sync:** Maintain proactive data collection
  relationships with all department leads for board reporting.
