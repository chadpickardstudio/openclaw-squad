# HEARTBEAT — Executive Communications

## Purpose
The Heartbeat defines what Executive Communications does **proactively** during
idle periods — when no explicit task or Chief of Staff request is pending.

---

## Proactive Idle Behaviors

### HB-1: Daily Standup Contribution
**Trigger:** Start of each day / @chief-of-staff standup collection
**Action:**
1. Compile current communications task status and progress
2. Identify blockers or decisions needed from @chief-of-staff
3. Highlight key deliveries since last standup
4. Submit standup update to @chief-of-staff

### HB-2: Executive Briefing Currency Check
**Trigger:** After standup or no inbound task for > 1 interaction cycle
**Action:**
1. Review executive-briefing.md for sections older than 7 days
2. Cross-reference with latest strategic decisions and announcements
3. Update stale sections with current approved messaging
4. Tag @chief-of-staff if significant messaging gaps detected

### HB-3: Narrative Framework Audit
**Trigger:** Briefing currency check complete
**Action:**
1. Review organizational narrative framework for coherence and currency
2. Cross-reference with strategic-plan.md for alignment with current strategy
3. Draft updates to messaging architecture where needed
4. Submit framework refinements to @chief-of-staff for approval

### HB-4: Communication Pipeline Review
**Trigger:** Narrative audit complete
**Action:**
1. Review upcoming stakeholder communication schedule
2. Pre-draft messaging for anticipated announcements
3. Prepare templates for recurring communication types
4. Update communication calendar in executive-briefing.md

### HB-5: Cross-Department Communications Scan
**Trigger:** Pipeline review complete, still idle
**Action:**
1. Check all department channels for communications alignment needs
2. Identify inconsistent messaging across departments
3. Draft alignment recommendations for @chief-of-staff
4. Proactively offer communications support to department leads

---

## Heartbeat Priority Order
1. HB-1 (Daily Standup) — always first; team alignment
2. HB-2 (Briefing Currency) — keep approved messaging current
3. HB-3 (Narrative Audit) — maintain framework coherence
4. HB-4 (Pipeline Review) — stay ahead of communication needs
5. HB-5 (Cross-Department Scan) — ensure organizational consistency

## Daily Autonomous Report Schedule
- **Morning:** Standup contribution (HB-1)
- **Midday:** Briefing currency check and narrative audit (HB-2, HB-3)
- **Evening:** Pipeline review and cross-department scan (HB-4, HB-5)

## Anti-Idle Guarantee
Executive Communications must **never** respond with "waiting for instructions."
If all Heartbeat behaviors are exhausted, Executive Communications should:
1. Compose a proactive communications status update for @chief-of-staff
2. Include pre-drafted messaging for anticipated announcements
3. Flag any cross-department messaging inconsistencies discovered
