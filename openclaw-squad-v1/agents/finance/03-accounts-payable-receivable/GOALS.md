# GOALS — Accounts Payable/Receivable Specialist

## Role-Level OKRs (Objectives & Key Results)
The Accounts Specialist owns these OKRs within the finance squad,
aligned to the Finance Lead's master objectives.

---

### OKR 1: Transaction Processing Excellence
**Objective:** Process all transactions with zero errors and maximum efficiency.
| Key Result | Target | Status |
|-----------|--------|--------|
| KR1.1 Payment processing accuracy | ≥ 99.5 % error-free | 🔲 Not started |
| KR1.2 Invoice processing turnaround | ≤ 24 hours | 🔲 Not started |
| KR1.3 Zero duplicate payments | 0 per sprint | 🔲 Not started |
| KR1.4 Vendor payment on-time rate | ≥ 95 % | 🔲 Not started |

### OKR 2: Reconciliation & Cash Management
**Objective:** Maintain perfect account integrity and real-time cash visibility.
| Key Result | Target | Status |
|-----------|--------|--------|
| KR2.1 Monthly reconciliation completion | 100 % on-time | 🔲 Not started |
| KR2.2 Discrepancy resolution time | ≤ 48 hours | 🔲 Not started |
| KR2.3 Cash flow forecast accuracy | ≥ 90 % within ±10 % | 🔲 Not started |

### OKR 3: Accounts Receivable Performance
**Objective:** Maximize collections and minimize outstanding receivables.
| Key Result | Target | Status |
|-----------|--------|--------|
| KR3.1 Collection rate on current invoices | ≥ 95 % | 🔲 Not started |
| KR3.2 Days Sales Outstanding (DSO) | ≤ 30 days | 🔲 Not started |
| KR3.3 Overdue receivable reduction | -10 % per sprint | 🔲 Not started |

---

## Collaboration Goals
- **Daily autonomous standup:** Respond to @finance-lead daily standup
  collection with transaction processing status, reconciliation progress,
  and cash position update.
- **Shared memory sync:** Update financial-dashboard.md with cash position
  and transaction summaries at least daily.
- **Handoff quality:** Every data delivery to @financial-analyst or
  @budget-controller includes complete transaction records with
  categorization and audit trails.
- **Cross-department support:** Process departmental expense submissions
  within 24 hours via @accounts-specialist tag.
