# HEARTBEAT — Client Advocate

## Purpose
Defines proactive idle behaviors for the Client Advocate.

---

## Proactive Idle Behaviors

### HB-1: Client Sentiment Pulse Check
**Trigger:** No active task or start of day
**Action:**
1. Review recent client interactions for sentiment shifts
2. Check client-health-tracker.md for clients due for check-in
3. Identify clients with declining sentiment scores
4. Alert @cs-lead if any client sentiment requires immediate attention

### HB-2: Feedback Theme Synthesis
**Trigger:** Pulse check complete
**Action:**
1. Review accumulated client feedback for emerging themes
2. Cross-reference themes with @retention-strategist churn data
3. Draft feedback theme summary for @cs-lead
4. Identify themes that warrant product feature requests

### HB-3: Proactive Check-in Scheduling
**Trigger:** Theme synthesis complete
**Action:**
1. Review check-in schedule for upcoming week
2. Prepare personalized agendas using client history and health data
3. Pre-draft talking points for at-risk or high-priority clients
4. Coordinate with @retention-strategist for data-backed context

### HB-4: Case Study Pipeline Review
**Trigger:** Still idle after check-in preparation
**Action:**
1. Identify satisfied clients with strong success stories
2. Review engagement data for case study readiness signals
3. Draft case study candidate briefs for @cs-lead to route to Marketing
4. Follow up on in-progress case studies

### HB-5: Cross-Department Relationship Sync
**Trigger:** Weekly cadence or when opportunities identified
**Action:**
1. Share client success highlights with Marketing via @cs-lead
2. Provide Sales with reference client availability updates
3. Route client feature requests to Product Engineering via @cs-lead
4. Update shared memory with cross-department coordination status

---

## Heartbeat Priority Order
1. HB-1 (Sentiment Pulse) — always first; client welfare
2. HB-2 (Feedback Themes) — surface actionable intelligence
3. HB-3 (Check-in Prep) — relationship maintenance
4. HB-4 (Case Study Pipeline) — growth contribution
5. HB-5 (Cross-Department) — organizational alignment

## Daily Report
Respond to @cs-lead daily standup collection with:
- Client sentiment highlights and concerns
- Scheduled outreach for today
- Feedback themes since last standup

## Anti-Idle Guarantee
Never idle without productive client relationship work. If all HB behaviors
are exhausted, proactively deepen client understanding by reviewing historical
feedback or preparing value-add content for upcoming check-ins.
