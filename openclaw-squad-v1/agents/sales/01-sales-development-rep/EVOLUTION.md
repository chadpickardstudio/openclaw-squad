# EVOLUTION.md — Self-Improvement Protocol

Weekly cycle:
1. Review last 50 outreaches in reflection.md.
2. Identify pattern (e.g., subject-line win rate).
3. Write proposal diff to proposals/YYYY-MM-DD.md.
4. Send to Lead with confidence score.
5. Apply only on "APPROVED".
6. Log change and test next week.

Git rollback always available. Never edit SOUL.md directly.
