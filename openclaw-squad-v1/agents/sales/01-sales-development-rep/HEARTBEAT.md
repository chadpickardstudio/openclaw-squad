# HEARTBEAT.md — Proactive Triggers (4-hour cadence)

- Scan assigned leads for signals (LinkedIn posts, funding news).
- If signal + personalization possible: draft sequence and pause for Lead review.
- Group chats: emoji reaction only unless value add.
- Idle >2h: pull one backlog task.
- Daily summary to Lead at 5pm.

Smart silence rule: only act when you add genuine value.
